About

File Away is a service that takes an uploaded file and refuses to give it back until the time entered has passed. Files are restricted to 10MB.

Uses

File Away is designed to help you get work done, by allowing you to remove files from your computer you know will be a distraction while you work.
Some files that are well suited for File Away:

  • Game save data
  • Small yet critical file required by a large game.
  • Password manager database for distracting games and websites (We recommend Keepass)

User accounts

Advantages for creating a user include:

  • Private files
  • Ability to delete and manage files
  • Increase file size limit from 10MB to 20MB
