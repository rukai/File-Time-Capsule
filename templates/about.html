About

File Time Capsule is a service that takes an uploaded file and refuses to give it back until the specified date. Files are restricted to 10MB.

Uses

File Time Capsule is designed to help you get work done, by allowing you to remove files from your computer you know will be a distraction while you work.
Some files that are well suited:

  • Game save data
  • Small yet critical file required by a large game.
  • Password manager database for distracting games and websites (We recommend Keepass)
