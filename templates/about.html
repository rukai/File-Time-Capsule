About

File Away is a simple service that takes an uploaded file and refuses to give it back until the time entered has passed.

Uses

File Away is designed to help you get work done, by allowing you to remove files from your computer you know will distract while you work.
Some files below 10MB that are well suited for File Away:

  • Game save data
  • Critical file required by a game
  • Textfile containing passwords to distracting websites

User accounts

Advantages for creating a user include:

  • Private files
  • Easily manage all your files
